# Отчёт: Нейронные сети и анализ тональности

## Архитектура сети

Реализованы две модели — для бинарной и мультиклассовой классификации.

**Бинарная** (положительный / отрицательный):
- Слои: Input → 64 нейрона → 32 нейрона → 1 выход
- Выход: Sigmoid (вероятность от 0 до 1)

**Мультиклассовая** (положительный / отрицательный / нейтральный):
- Слои: Input → 128 → 64 → 32 → 3 выхода
- Выход: Softmax (вероятности для каждого из 3 классов)

Все скрытые слои используют активацию **ReLU**. Данные предварительно векторизуются с помощью **TF-IDF** (уни- и биграммы, 200 признаков).

---

## Как работает Forward Pass

Прямой проход — это последовательные вычисления через каждый слой сети:

1. Берём входные данные (TF-IDF вектор отзыва)
2. Умножаем на матрицу весов слоя, прибавляем смещение: `z = X @ W + b`
3. Применяем функцию активации: `a = ReLU(z)`
4. Передаём результат в следующий слой
5. На последнем слое вместо ReLU используем Sigmoid или Softmax — они дают вероятности классов

---

## Как работает Backpropagation

После того как сеть сделала предсказание, нужно посчитать, насколько оно ошиблось, и скорректировать веса.

1. Считаем ошибку на выходе: разница между предсказанием и правильным ответом
2. «Передаём» эту ошибку назад через все слои — от выходного к входному
3. По пути считаем, как каждый вес повлиял на ошибку (градиент)
4. Обновляем веса в сторону уменьшения ошибки: `W = W - lr · градиент`

Этот процесс повторяется для каждого батча данных на каждой эпохе обучения.

---

## Dropout

Dropout — это способ борьбы с переобучением. Во время обучения случайные 30% нейронов скрытых слоёв обнуляются на каждом шаге. Это заставляет сеть не «запоминать» конкретные примеры, а учиться более общим закономерностям.

При предсказании (inference) dropout отключается — все нейроны работают в полную силу.

---

## Early Stopping

Чтобы сеть не переобучилась, используется ранняя остановка. После каждой эпохи проверяем ошибку на валидационной выборке. Если она не улучшается 20–25 эпох подряд — обучение останавливается, а веса возвращаются к лучшему найденному состоянию.

## Результаты

| Задача | F1-score | Precision | Recall |
|---|---|---|---|
| Бинарная (2 класса) | ~0.90 | ~0.90 | ~0.90 |
| Мультиклассовая (3 класса, macro) | ~0.80 | ~0.80 | ~0.80 |

Бинарная задача решается проще — слова «ужасный» и «отличный» хорошо разделяют классы. Нейтральные отзывы сложнее — их лексика менее выразительна, поэтому точность чуть ниже.

## Вывод

Нейронная сеть с нуля на NumPy успешно справляется с задачей анализа тональности. Dropout и Early Stopping помогают избежать переобучения на небольшом датасете. При увеличении объёма данных качество модели будет выше и стабильнее.
